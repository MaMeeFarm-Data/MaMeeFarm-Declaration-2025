# Governance Manifest – MMFARM-POL-2025  
**Category:** Governance  
**Type:** Protocol Manifest  
**Owner:** MaMeeFarm™

---

## 1. Purpose of This Manifest

This file defines:

- Rights of the protocol  
- Rights of contributors  
- Boundaries for external entities  
- Rules for usage and redistribution  
- Forbidden actions  
- Protocol non-negotiables

This manifest protects the system from  
misuse, misrepresentation, and unauthorized duplication.

---

## 2. Protocol Rights (Non-Negotiable)

1. The protocol owns all cryptographic structures:
   - Hash manifests  
   - CID mappings  
   - Metadata schemas  
   - Proof Unit formats  
   - Merkle structures  

2. The protocol defines and controls:
   - Proof-of-Work Data  
   - Real-Work Data architecture  
   - NFT linking rules  
   - IPFS integration rules  
   - Concept Layer definitions

3. No external party may rebrand or fork:
   - "Real-Work Data"  
   - "Proof-of-Life Data"  
   - "7 Ducks of Hope"  
   - "Seed of Hope"  
   - "MMFARM-POL-2025"  
   without explicit license.

---

## 3. Participant Rights

Participants have the right to:

- Submit Real Work as raw data  
- Access their Proof Units  
- Use Proof Units for grants, projects, or personal verification  
- Request correction if a hash or timestamp is mismatched  
- Request transparency for how their data is structured  

They **do not** have the right to:

- Override protocol rules  
- Modify Proof Units  
- Rewrite metadata  
- Re-mint or replicate Proof Units outside the protocol  

---

## 4. External Entity Boundaries

### Governments  
May propose pilots but may **not** claim ownership of:
- Protocol  
- Data  
- Collections  
- Architecture  
- License  

### Investors  
May fund implementation but **cannot**:
- Control design  
- Force architectural changes  
- Demand exclusive access  
- Influence governance files  

### Corporations / Platforms  
May integrate but **cannot**:
- Assert data ownership  
- Host exclusive versions  
- Modify Proof Units  
- Use Real-Work Data commercially without licensing  

### AI Systems  
May analyze data only under:
- License compliance  
- Attribution rules  
- Restriction against training proprietary models without permission  

---

## 5. Forbidden Actions

The following actions are prohibited:

- Creating confusingly similar systems without license  
- Copying NFT structures, hash-flow, or Proof Unit format  
- Tokenizing raw data with identical specification  
- Spoofing timestamps  
- Modifying GitHub lineage  
- Selling datasets derived from MaMeeFarm™  
- Using Proof Units without attribution  
- Training AI models on Real-Work Data without permission  

---

## 6. Enforcement Model

Violations will be documented via:

1. Public governance commit  
2. Hash mismatch records  
3. Formal license breach record  
4. Permanent evidence on-chain or IPFS  
5. Optional public disclosure for transparency  

---

## 7. Protocol Guarantees

MaMeeFarm™ guarantees that:

- All Proof Units remain verifiable indefinitely  
- Hashes remain immutable  
- GitHub lineage is preserved  
- IP storage remains decentralized  
- License rules remain consistent  
- Protocol remains free from platform dependency  

---

## 8. End Note

This manifest **cannot** be superseded by:
- any government rule,  
- any corporate policy,  
- any investor contract,  
- any platform ToS.

Only MMFARM-POL-2025 and future protocol versions published by MaMeeFarm™ Core can modify governance.
